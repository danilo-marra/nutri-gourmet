# UI/UX — Diretrizes Visuais

**Summary**: Padrões de interface definidos no PRD para o sistema de gestão de cantina.

**Sources**: raw/prd.md

**Last updated**: 2026-05-27

---

O PRD define um estilo visual corporativo moderno com foco em valor percebido. (source: raw/prd.md)

## Diretrizes

- **Estilo**: corporativo moderno
- **Fundo**: claro
- **Paleta**: azul e cinza como cores principais
- **Ícones**: discretos
- **Layout**: focado em valor percebido — clareza e eficiência acima de ornamentação

## Intenção

A interface deve transmitir confiabilidade e profissionalismo para a Gestora ([[administrador]]), ao mesmo tempo que é ágil e direta para o [[operador]] no fluxo de vendas diárias.
